### 👋 Hi, I'm Andreas

I use GitHub to share hobby projects that I build mainly for fun and out of curiosity.

Most of these projects start as experiments, ideas I want to understand better, or things I simply enjoy tinkering with. When it makes sense, I publish them under the MIT License so others can explore, reuse, or learn from them as well.

In many ways, this is my personal space for reinventing the four-sided wheel — solving problems that may already have solutions, just to see how they work and what I can learn along the way.
